📄 2. requirements.txt
flask
gunicorn

3. Procfile
web: gunicorn app:app

STEP 4: Create Repository on GitHub
Go to GitHub
Click New Repository
Name: flask-app
Click Create Repository

STEP 5: Upload Files (No Command Prompt)
Open your repo
Click Add file → Upload files
Upload:
app.py
requirements.txt
Procfile
Click:
Commit changes

STEP 6: Connect to Render
Open Render
Click GitHub → Authorize
Allow access
Select your repo: flask-app

🟢 STEP 7: Deploy Application
Fill details:
Environment: Python
Build Command: pip install -r requirements.txt
Start Command: gunicorn app:app
Click:
Create Web Service

🟢 STEP 8: Wait for Deployment
Render will:
Install packages
Build app
Start server
⏳ Wait 1–2 minutes


🟢 STEP 9: Run Your Web App
Render gives URL like:
https://flask-app.onrender.com
👉 Open it in browser.
